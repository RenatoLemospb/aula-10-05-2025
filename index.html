Primeiros Passos com Git

Publicado em: 10 de Maio de 2025

Nesta aula, aprendemos os conceitos básicos do Git, como inicializar um repositório com git init, adicionar arquivos com git add e criar commits com git commit. É o início do controle de versão!

Trabalhando com Branches

Publicado em: 8 de Maio de 2025

Exploramos como criar e gerenciar branches no Git usando git branch e git checkout. Branches são essenciais para trabalhar em diferentes funcionalidades sem afetar o código principal.
